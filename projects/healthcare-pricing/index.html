The project

Useful estimates without looking into the future.

CareLens is an end-to-end machine learning application that estimates hospital costs from demographic, clinical, and admission-time information while making uncertainty and model limitations visible.

248patient records
18admission-time features
27.4%lower held-out MAE

Preventing target leakage

I excluded variables that only become available after admission so the model reflects a realistic prediction setting. Reusable preprocessing pipelines handle missing-value imputation, categorical encoding, feature scaling, and log-transformed regression consistently across training and inference.

Evaluation against a real baseline

I compared Ridge regression and Random Forest models with a median baseline using five-fold cross-validation. The selected workflow reduced held-out mean absolute error by 27.4%, from ₹83.7K to ₹60.7K.

Reliability and limitations

Permutation importance, empirical prediction intervals, subgroup error analysis, and data-quality checks make the model easier to audit. The model card documents important limitations, including the small dataset and higher uncertainty for unusually expensive cases.

What I delivered

  • A reproducible, leakage-aware training and preprocessing pipeline
  • Cross-validated model comparison against a median baseline
  • Prediction intervals, subgroup diagnostics, and data-quality checks
  • Automated tests, reproducible training scripts, and a model card
  • A deployed Streamlit application for interactive cost estimates
